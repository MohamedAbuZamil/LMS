# Teacher Frontend — LMS

لوحة تحكم المدرس لإدارة الكورسات، الحصص، الفيديوهات، الامتحانات، بنوك الأسئلة، السكرتارية، الطلاب، الحضور، والتقارير.

## التشغيل

```bash
npm install
npm run dev
```

يفتح على <http://localhost:3002>

**حساب تجربة (مدرس):** `ahmed` / `123456`

## الأقسام الرئيسية

- **اللوحة:** نظرة عامة وإحصائيات.
- **الكورسات:** إضافة/عرض كورسات المدرس مع الحصص والأسعار.
- **بنوك الأسئلة:** إنشاء بنوك وإضافة أسئلة (اختيار متعدد / صح وخطأ / مقالي).
- **إدارة الحصص:** كورس ← حصة ← (فيديوهات — امتحان — ملفات).
- **الامتحانات:** بانٍ أسئلة يدوي + عشوائي من بنوك.
- **الطلاب:** إضافة بكود الطالب مع تسجيل الدفع وفلتر المحافظات.
- **الحضور:** متابعة حضور وغياب + إشعار ولي الأمر.
- **السكرتارية:** إدارة صلاحيات الحسابات المساعِدة.

> ملاحظة: البيانات حالياً mock في `sessionStorage`. عند الربط الحقيقي ستتبدل طبقة الوصول للبيانات في `src/lib/data/*` بطلبات API.
